在linux中安装mysql

发表于2020-11-17|更新于2020-11-18|mysqlcentos
|阅读量:

说明

1.本次采用的环境是VM+centos7进行的。
2.mysql安装包可到Oracle官网上下载。

步骤说明

解压及创建目录

1.tar -xzvf mysql-5.7.11-linux-glibc2.5-x86_64.tar.gz -C /usr/local
2.cd /usr/local
3.mv mysql-5.7.11-linux-glibc2.5-x86_64 mysql
4.mkdir mysql/arch mysql/data mysql/tmp

创建my.cnf

1.vi /etc/my.cnf
2.将文件内容复制到你的文件中，并保存

创建用户组及用户

1.groupadd -g 101 dba #创建组并指定组id
2.useradd -u 514 -g dba -G root -d /usr/local/mysql mysqladmin #创建用户并指定组
3.id mysqladmin #检查是否创建成功

copy 环境变量配置文件至mysqladmin用户home目录

1.cp /etc/skel/.* /usr/local/mysql #为了以下步骤配置个人环境变量

配置环境变量

1.vi mysql/.bashrc #也可以为mysql/.bash_profile,但不建议
将以下内容复制到配置文件的末尾

赋权限和用户组，切换用户mysqladmin，安装

1.chown mysqladmin:dba /etc/my.cnf
2.chmod 640 /etc/my.cnf
3.chown -R mysqladmin:dba /usr/local/mysql
4.chmod -R 755 /usr/local/mysql

配置服务及开机自启动

1.cd /usr/local/mysql
#将服务文件拷贝到init.d下，并重命名为mysql
2.cp support-files/mysql.server /etc/rc.d/init.d/mysql
#赋予可执行权限
3.chmod +x /etc/rc.d/init.d/mysql
#删除服务
4.chkconfig –del mysql
#添加服务
5.chkconfig –add mysql
6.chkconfig –level 345 mysql on
#添加开机启动命令
7.vi /etc/rc.local 并将以下文字拷贝至末尾

#给启动文件赋予执行权限，注意/etc/rc.local只是个软件链接，要给真实的文件赋予执行权限
8.chmod +x /etc/rc.d/rc.local

安装libaio及安装mysql的初始db

1.yum -y install libaio
2.sudo su - mysqladmin
3.bin/mysqld \
–defaults-file=/etc/my.cnf \
–user=mysqladmin \
–basedir=/usr/local/mysql/ \
–datadir=/usr/local/mysql/data/ \
–initialize

查看临时密码

1.cat /usr/local/mysql/data/hostname.err | grep password
A temporary password is generated for root@localhost: 4Qiz%U;3.ohf

启动

1./usr/local/mysql/bin/mysqld_safe –defaults-file=/etc/my.cnf &

登录及修改用户密码

1.mysql -uroot -p’4Qiz%U;3.ohf’ #使用查看得到的临时密码
#进入服务后
2.mysql> alter user root@localhost identified by ‘m123456’; #指定你的密码
3.mysql> GRANT ALL PRIVILEGES ON . TO ‘root’@’%’ IDENTIFIED BY ‘m123456’ ;
#刷新
4.mysql> flush privileges;
#退出服务
5.mysql> exit;

重启

1.service mysql restart

.
